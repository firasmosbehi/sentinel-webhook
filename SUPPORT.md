# Support

## Where To Ask Questions

- GitHub Issues: for bugs and feature requests.
- GitHub Discussions (recommended): for Q&A, how-to, and announcements.

## What To Include In A Bug Report

- The input used (sanitized).
- The target URL and whether `rendering_mode` is `static` or `playwright`.
- Any relevant run dataset items (especially `event_id` and error messages).
- Expected behavior vs actual behavior.

## Security Issues

Please do not report security issues in public issues.

See `SECURITY.md`.
